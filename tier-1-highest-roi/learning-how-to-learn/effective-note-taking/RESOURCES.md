# Resources: Effective Note-Taking

## Primary Sources

- **How to Take Smart Notes** — Sönke Ahrens (book)
  The definitive guide to the Zettelkasten method. Core argument: notes are for thinking, not storage.

- **Cornell Note-Taking System** — Walter Pauk, Cornell University
  https://lsc.cornell.edu/how-to-study/taking-notes/cornell-note-taking-system/
  Original structured format: cues, notes, summary. Free and well-evidenced.

- **Make It Stick** — Brown, Roediger, McDaniel (book)
  Evidence-based learning science. Relevant chapters: interleaving, elaborative interrogation, the testing effect.

## Secondary Sources

- **The Learning Scientists** — https://www.learningscientists.org/
  Research-backed learning strategies explained in plain English. Useful for citations.

- **Tiago Forte — Building a Second Brain** (book/blog)
  CODE method: Capture, Organise, Distil, Express. Practical modern framework.
  https://fortelabs.com/blog/basboverview/

## Communities

- **r/Zettelkasten** — https://reddit.com/r/Zettelkasten — active community for linking note methods
- **r/productivity** — broad note-taking discussions and system comparisons
