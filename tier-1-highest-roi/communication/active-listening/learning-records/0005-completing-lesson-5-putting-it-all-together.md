# Putting It All Together — The Complete Listening Loop

User completed Lesson 5 (final lesson in the Active Listening series). The exercise: a simulated 1:1 meeting transcript with four decision points, each mapping to one of the four learned skills.

The complete listening loop now established:
1. **Notice the gap** (Lesson 1) — catch what you're filling your spare mental capacity with (interrupting, zoning, solution-jumping).
2. **Reflect** (Lesson 2) — paraphrase what the speaker meant before responding.
3. **Ask** (Lesson 3) — if still unclear, ask an opening question (not an agenda question).
4. **Re-engage** (Lesson 4) — if you drifted, use the Last Word Anchor to re-enter cleanly.

The loop is not always sequential. Reflection alone is often enough. The question only follows when something genuine is still unclear. Re-engagement is triggered by drift, not every turn.

**Active Listening series complete.** User has a full, integrated listening loop that can be practiced one skill at a time in real meetings until all four become automatic.
