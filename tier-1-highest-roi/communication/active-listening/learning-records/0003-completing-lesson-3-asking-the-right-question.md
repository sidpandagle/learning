# Asking the Right Question — Opening vs. Agenda Questions

User completed Lesson 3. The core skill: after reflecting, the next move is either "that's enough, respond" or "I still don't fully understand — ask an opening question."

Key distinctions now established:
- **Opening questions** invite the speaker to go further in *their* direction — you learn something you didn't already know.
- **Agenda questions** steer toward a conclusion you've already reached, even when phrased as curiosity (e.g., "Don't you think…", "Wouldn't it make more sense to…", "Have you considered…").
- The tell: if your preferred answer is embedded in the question, it's not really a question.

The full listening sequence so far:
1. **Reflect** — paraphrase what they meant before responding
2. **Ask** — if still unclear, ask a question that opens, not one that redirects

Both moves aren't always needed: if the reflection is confirmed, move on. Use the question only when something genuine is still unclear.
