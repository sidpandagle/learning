## What was established
User is starting from scratch with no existing note-taking system. Notes are captured randomly or not at all.

## Key context
- Goal: build a consistent system, not learn a sophisticated PKM tool
- Primary pain: no system exists — first lessons should focus on the *why* and a minimal viable method
- Linked to: building-systems-for-continuous-learning workspace (already completed lessons on forgetting curve, active recall, spaced repetition)

## Zone of proximal development
Start with: why passive note-taking fails → the one core principle (notes as thinking, not copying) → a simple structured format (Cornell or equivalent).

Do not start with: Zettelkasten, linking notes, tagging systems, or tool choice. These are premature.
