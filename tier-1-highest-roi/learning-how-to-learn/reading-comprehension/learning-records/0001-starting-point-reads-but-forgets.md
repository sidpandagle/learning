## What was established
User reads actively but retention is the core problem — ideas feel understood in the moment but are gone within a day.

## Key context
- Not a speed or decoding problem — the bottleneck is retention after reading
- Has completed the forgetting curve and active recall lessons (parent workspace) — so the *why* behind retention is already understood
- First lessons should focus on *how to read differently*, not on post-reading review (that's covered elsewhere)

## Zone of proximal development
Start with: the difference between passive and active reading → the pre-reading Survey + Question phase (SQ3R) → reading with questions as a frame.

Do not start with: annotation systems, post-reading summaries, or speed-reading. Address the reading *process* first.
