# Resources: Research Skills

## Primary Sources

- **Web Literacy for Student Fact-Checkers** — Mike Caulfield (free ebook)
  https://webliteracy.pressbooks.com/
  The SIFT method and lateral reading. The best practical guide to evaluating online sources.

- **Calling Bullshit** — Carl Bergstrom & Jevin West (book)
  How to spot misleading data, statistics, and claims. Chapter 2 on source evaluation is particularly strong.

- **CRAAP Test** — Meriam Library, CSU Chico
  Currency, Relevance, Authority, Accuracy, Purpose. Classic source evaluation checklist.
  https://library.csuchico.edu/help/source-or-information-good

## Secondary Sources

- **Google Scholar** — https://scholar.google.com/
  For finding academic primary sources on any topic.

- **Our World in Data** — https://ourworldindata.org/
  Reliable, well-cited data visualisations on global topics. Good model for what trustworthy data presentation looks like.

- **Snopes** — https://www.snopes.com/
  Fact-checking site. Useful both for checking claims and for understanding *how* fact-checkers work.

## Communities

- **r/educationalmedia** — Discussion of media literacy and source evaluation
- **r/skeptic** — Critical thinking community; frequent discussions on evaluating evidence
