# Starting point: no prior learning system

User is starting from zero — has not tried any structured learning technique (no Anki, no Zettelkasten, no Feynman technique, no spaced repetition). Goals are broad: wants to learn and grow generally, not tied to a specific domain. No time or budget constraints.

**Implications:** Begin with fundamentals of learning science (forgetting curve, active recall) before introducing any tools or systems. Keep early lessons short and concrete. The learning system should be built from first principles, not tool-first.
